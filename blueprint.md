# **App Name**: 404 found

## Core Features:

- Informational Sections: Dedicated sections for describing Web Development and IoT & Home Automation services.
- Contact Form: Collect user information (Name, Email, Phone, Message, Service Interest) and project details.
- Email Integration: Automatically send form submissions to the provided email address.
- Google Sheets Integration: Automatically save form submission data to a Google Sheets spreadsheet via Google Forms / Sheets API / Zapier.
- SEO Optimization: Implement meta title and description to improve search engine visibility.
- Chat Assistant: AI chatbot answers common questions from businesses and homeowners. Uses tool to decide when and how to include information about the specific region of India that the user is in.

## Style Guidelines:

- Primary color: Deep blue (#1E3A8A) for a professional and trustworthy feel.
- Background color: Light gray (#F5F5F5) to provide a clean and modern look.
- Accent color: Teal (#2DD4CF) to highlight important calls to action and interactive elements.
- Headline font: 'Poppins' (sans-serif) for headlines to create a modern, avant-garde look.
- Body font: 'PT Sans' (sans-serif) for body text to provide clarity and readability.
- Use modern, line-style icons to represent web development and IoT services.
- Maintain a clean and well-structured layout with clear visual hierarchy and mobile responsiveness.
